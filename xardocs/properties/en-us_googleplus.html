Description

The Google Plus dynamic data property allows you to easily add a Google +1 button on your site for any page or item and customise it with a number of standard button attributes.

On output, the Google Plus button will display the standard Plus One icon button, and can also display counts and customisable text.

There is no value saved for this Dynamic Data property and no input page. All customisation is in the configuration of the property itself or passed into the tag.

Attributes

  • annotation: [default='bubble']
    Determines the style of the button with bubble with count as default. Can be 'bubble,'inline', or 'none'.
  • button size: [default='standard']
    Size of the google button which can be either 'small', 'standard', 'medium', or 'tall'.
  • displaywidth: [default=450]
    Width in pixels of the widget.
  • plusurl: [default= current page URL]
    Will default to the current page URL when left empty.

Other attributes may be inherited from the parent property.

Usage

Standalone

  • In your template forms: you can use the data-output tag to display the Google plus button. 
        <xar:data-display type="googleplus" annotation="bubble" 
            [attr1="value1",...] />

Defined Data Property

  1. Defining your data property: You can define the attributes in the Configuration screen. To display the property independently, use the data-display tag anywhere you wish the property to display or hook dynamic data to your module/itemtypes for automatic display. 
        <xar:data-input property="$properties['googleplus']"
            annotation="bubble" [attr1="value1",...] />
