Description

The Facebook Like dynamic data property allows you to easily add a Facebook button on your site for any page or item and customise it with a number of standard Facebook button attributes.

On output, the Facebook Like button will display the standard Facebook Like icon button, and can also display send button, counts and customisable text.

There is no value saved for this Dynamic Data property and no input page. All customisation is in the configuration of the property itself or passed into the tag.

Attributes

  • layoutstyle: [default='standard']
    Determines the style of the button with bubble count or box with count, and other text and can be either 'standard', 'button_count', 'box_count'.
  • scheme: [default='light']
    A dark or light color scheme for the button.
  • sendbutton: [default=false]
    When true will display a send button next to the like button.
  • showfaces: [default=false]
    When true will display the profile picture below the link button.
  • verbtext: [default=Like]
    Determines the text to display in the Facebook button.
  • likeurl: [default= when empty uses the current page url]
    The URL that will be submitted to Facebook with the button submit.
  • displaywidth: [default= 450]
    Width in pixels of the widget.

Other attributes may be inherited from the parent property.

Usage

Standalone

  • In your template forms: you can use the data-output tag to display the Facebook button. 
        <xar:data-display type="facebooklike" layoutstyle="button_count" 
            [attr1="value1",...] />

Defined Data Property

  1. Defining your data property: You can define the attributes in the Configuration screen. To display the property independently, use the data-display tag anywhere you wish the property to display or hook dynamic data to your module/itemtypes for automatic display. 
        <xar:data-input property="$properties['facebooklike']"
            layoutstyle="button_count" [attr1="value1",...] />
